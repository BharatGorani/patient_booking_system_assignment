# Patient Appointment Booking System

## Overview

This is a full-stack web application for managing patient appointments. Patients can register, login, book, reschedule, and cancel appointments. Doctors/Admins can manage appointments and update their statuses.

---

## Technologies Used

- Backend:
  - Node.js
  - Express.js
  - MongoDB with Mongoose
  - JWT Authentication

- Frontend:
  - React.js with Context API and React Router

---
